About This Project
This project is a static web page that serves as a comprehensive resource for journalists, researchers, and activists interested in using digital tools for environmental investigations.
The website summarizes the key techniques, workflows, and tools presented during the panel "How Digital Tools Help Uncover Environmental Stories" at the Global Investigative Journalism Conference 2025 (GIJC25). It aims to make the valuable information from this session easily accessible, well-organized, and persistent for future reference.
Content Source
The content for this website is entirely based on the public materials provided for the GIJC25 panel. The primary resources were a presentation and a detailed tipsheet created by Reinaldo Chaves of Abraji (Brazilian Association of Investigative Journalism).

Key Features of the Website
Panel Summary: Provides context about the GIJC25 session and introduces all the expert panelists.
Core Investigative Techniques: A detailed breakdown of the five essential methods for modern environmental investigation, from database linking to network analysis.
Categorized Tool Directory: A curated list of recommended digital tools, grouped by function (e.g., Satellite Analysis, Data Platforms, Visualization) for easy reference.
Key Takeaways: Highlights the three main principles for reporters to remember when tackling environmental stories.
Responsive Design: A clean, modern, and fully responsive layout ensures the content is easily readable on any device.
Direct Links to Sources: Includes prominent buttons and footer links to the original presentation, tipsheet, and official conference page.
